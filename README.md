#  Customer Churn Analysis & Prediction

##  Project Overview
Customer churn is a major challenge for subscription-based businesses.  
This project analyzes customer behavior to identify churn drivers and predicts which customers are likely to churn using machine learning, enabling proactive retention strategies.

---

##  Business Objective
- Understand **why customers churn**
- Identify **key churn drivers**
- Predict **high-risk customers**
- Support **data-driven retention decisions**
